# Tavern Helper Docs Index

这个目录存放从 `tavern_helper_template-main` 移入的教程、规则、类型定义和示例，方便后续维护 SillyView 时直接在项目内读取。

常用入口：

- `CLAUDE.md`: 原模板的总入口说明。
- `.cursor/rules/酒馆助手接口.mdc`: TavernHelper 接口说明入口。
- `.cursor/rules/项目基本概念.mdc`: 项目结构和基础概念。
- `.cursor/rules/前端界面.mdc`: 前端界面编写规则。
- `.cursor/rules/脚本.mdc`: 脚本编写规则。
- `.cursor/rules/酒馆变量.mdc`: 酒馆变量说明。
- `.cursor/rules/mvu变量框架.mdc`: MVU 变量框架说明。
- `.cursor/rules/mvu角色卡.mdc`: MVU 角色卡说明。
- `@types/function/generate.d.ts`: AI 生成接口类型。
- `@types/function/worldbook.d.ts`: 世界书接口类型。
- `@types/function/`: TavernHelper 函数类型定义。
- `@types/iframe/`: iframe 环境导出类型定义。
- `slash_command.txt`: 斜杠命令资料。
- `示例/`: 原模板示例。
- `初始模板/`: 原初始模板资料。
- `init/`: 初始化文档和示例。
